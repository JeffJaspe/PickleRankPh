# Pickleball PH

Production-ready ranking platform for Philippine Pickleball.

## Setup
1. npm install
2. Copy .env.example to .env.local and fill in Supabase values
3. Run supabase/schema.sql and supabase/seed.sql in Supabase SQL editor
4. npm run dev

## Deploy
Push to GitHub, then import to Vercel. Add env vars in Vercel dashboard.
